Summary: Latitude vs. X

The purpose of this project was to analyze how weather (specifically temperature, cloudiness, wind speed, and humidity) changes relative to your position to the equator. To accomplish this task, data was pulled from the OpenWeatherMap API to create a dataset covering over 500 cities.

Matplotlib library was used to create plots of latitudes against already identified specific variables. This site provides the source data and visualizations created as part of the analysis, as well as explanations for the variation of weather across lititudes.
